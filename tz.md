# Техническое задание на разработку сайта MARMELAT

## 1. Общая информация о проекте

### О бренде
**MARMELAT** — казахстанский бренд культовых средств для укладки волос.

**Миссия:** Помочь каждой женщине почувствовать себя настоящей звездой, предоставляя продукты и инструменты для создания идеальных укладок и средств для волос, которые подчеркнут её внутреннюю силу, красоту и уверенность в себе.

**Слоган:** «Создавая образ — выражай свою индивидуальность»

**Целевая аудитория:** Женщины, заинтересованные в уходе за волосами и создании стильных укладок (как домашнее использование, так и профессионалы — парикмахеры и стилисты).

### 3.3 Каталог продукции

Разделить на подсекции с карточками товаров:

#### 3.3.1 Восковой стик для укладки волос

**Заголовок:** `Восковой стик MARMELAT®`

**Описание:**
```
Создан для создания гладких и блестящих причёсок. Благодаря формату стика, 
воск легко распределяется, не утяжеляя волосы, его удобно наносить и 
контролировать объём применяемого средства для волос.

Подходит для безупречных укладок в домашних условиях, а также парикмахерам 
и стилистам. На волосах оставляет глянцевый эффект без жирности.
```

**Состав (ключевые ингредиенты):**
- **Канделильский воск** — увлажнение без склеивания и утяжеления, защитный барьер
- **Пчелиный воск** — натуральный увлажнитель, придаёт плотность и текстуру
- **Касторовое масло** — сияние и блеск, предотвращает ломкость, укрепляет волосы

**Характеристики:**
- Объём: 50 ml
- Веганская формула
- Не тестируется на животных
- Подходит для всех типов волос

**Цена:** РРЦ 8 000 тг. / ОПТ 5 200 тг.

**Изображение:** Несколько стиков на подставке (слайд 3)

## 4. Технические требования

### Адаптивность
- Desktop: 1920px, 1440px, 1280px
- Tablet: 768px, 1024px
- Mobile: 375px, 414px

### Анимации
- Плавное появление секций при скролле (fade-in, slide-up)
- Hover-эффекты на карточках товаров (увеличение, тень)

### Интерактивные элементы
- Sticky-навигация с прозрачностью → цветом при скролле
- Lightbox для увеличения фото продуктов
- Плавный скролл к секциям
- Кнопка "Наверх"

### SEO
- Семантическая HTML-разметка
- Alt-теги для всех изображений
- Meta-теги (title, description)
- Open Graph для соцсетей

### Производительность
- Lazy loading для изображений
- Оптимизированные изображения (WebP с fallback)
- Минификация CSS/JS

## 9. Примечания для Claude Code

1. **Используй CSS-переменные** для цветов и шрифтов
2. **Создай компонентную структуру** (каждая секция — отдельный блок)
3. **Добавь плейсхолдеры для изображений** с указанием размеров
4. **Реализуй smooth scroll** для навигации
5. **Сделай полностью адаптивный дизайн** (mobile-first)
6. **Добавь микро-анимации** для интерактивности
7. **Используй Grid и Flexbox** для layout
8. **Комментируй код** для удобства поддержки
