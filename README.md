# MARMELAT

Лендинг и инвестиционная презентация для казахстанского бренда средств для укладки волос MARMELAT®.

## Технологии

- **React 18** + **TypeScript** + **Vite**
- **TailwindCSS** — стилизация с кастомными цветами бренда
- **Framer Motion** — анимации и интерактивность
- **React Router** — маршрутизация

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера (порт 3000)
npm run dev

# Сборка для продакшена
npm run build

# Превью продакшен-сборки
npm run preview

# Линтинг
npm run lint
```

## Лендинг

Одностраничный сайт с секциями:
1. **Header** — фиксированная навигация
2. **Hero** — главный экран с CTA
3. **About** — миссия бренда
4. **Benefits** — 6 преимуществ (натуральный состав, cruelty-free, веган и др.)
5. **Products** — каталог из 5 товаров
6. **Bundles** — 2 готовых набора
7. **Partners** — 7 партнёров (Magnum, Золотое Яблоко, Cosmetica_KZ и др.)
8. **Footer** — контакты и соцсети

### Товары

- Восковой стик
- Гель для пушковых волос
- Щётка и расчёска-гребень
- Профессиональная щётка
- Шёлковые бигуди

Цены в тенге (KZT) с двумя вариантами: розничная и оптовая.

## Питч-дек

12-слайдовая презентация с навигацией клавишами (стрелки, пробел) и свайпами:

1. Cover — заголовок, Seed Round $2M
2. Problem — анализ рынка
3. Solution — позиционирование MARMELAT
4. Market — TAM/SAM/SOM
5. Products — 8 SKU, категории
6. Business Model — B2B + D2C
7. Traction — метрики ($100K выручка, 50K+ Instagram)
8. Competition — матрица конкурентов
9. Team — команда
10. Financials — прогнозы, unit economics
11. Investment — использование средств
12. Contact — контакты

### Графики

- **RevenueChart** — линейный график выручки
- **DonutChart** — круговые диаграммы
- **CompetitiveMatrix** — scatter-plot конкурентов
- **UnitEconomicsDashboard** — метрики CAC, LTV
